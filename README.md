# CookFlow 🍳

Sistema de gestão de pedidos ultra-rápido para restaurantes, construído com HTML, CSS, JavaScript e Supabase.

## Como configurar

1. **Supabase**:
   - Crie um projeto no [Supabase](https://supabase.com).
   - Vá em **SQL Editor** e execute o conteúdo do arquivo `setup.sql`.
   - Vá em **Project Settings > API** e copie a `Project URL` e a `anon public key`.
   - No arquivo `supabase.js`, substitua `SUA_SUPABASE_URL_AQUI` e `SUA_SUPABASE_ANON_KEY_AQUI` pelas suas chaves.

2. **Deploy na Vercel**:
   - Basta subir estes arquivos para um repositório no GitHub.
   - Conecte o repositório na [Vercel](https://vercel.com).
   - O deploy será automático!

## Funcionalidades
- Interface otimizada para tablets e totens.
- Atualização em tempo real (Realtime).
- Design moderno e minimalista com a identidade CookFlow.
- Gestão de fluxo: Pendente -> Preparando -> Pronto -> Entregue.
